Về project Ace

Hiện tại em đang chạy theo quy trình sau:

1. Download file tại thư mục /ace/files/Dados/_OK/
Mình dùng các file bắt đầu bằng call_….xslx

2. Tạo một database bằng postgressSQL tạo một table để chưa call log

3. Dùng nodejs để import data vào anh tham khảo file import_file.js

4. Tạo view tổng hợp data. Anh tham khảo file call.sql

5. Export data của view trên ra file processed_data.csv

6. Dùng code python ở file Main.ipynb  chạy bằng jupyter notebook để tính forecast và ghi ra file theo yêu cầu

Các file em vừa nói đều ở github 

https://github.com/anhphamkstn/timesheet
